Steps to use this project effectively

  1. Open index.ts and comment out the imports from ./answers
  2. Comment out the now undefined functions to get rid of the errors
  3. Run yarn start
  4. Open the console/dev tools
  5. Add your code for each of the problems

If you follow these steps you should be able to avoid seing the answers to the questions before attempting to solve them yourself.

The answers can be found in ./answers.ts.

Contributing to this repo

If you wish to add new problems or answers, and/or if you think some existing problems or answers need changes, you are more than welcome to submit a pull request.
